Sketch Data Collection Application

This is an example data collection application built for researchers who need a sketch data collection interface.

There is a drawing canvas on the next page that collects the drawings in the following format:

[ [[x_points], [y_points]], ..., [[x_points], [y_points]] ]

The last page takes the credentials of the participant and sends the drawings to Firebase realtime database.

**Please use the same device until completing the drawings.
